##git上传代码到github

假如要上传README.md，首先在该文件所处的文件夹下打开git bash，后面如下

1）git init初始化当前文件夹

2）git add .（添加文件到本地库.表示当前文件夹下的所有文件，也可以单独的文件git README.md）

3）git status（可要可不要：查看当前的git状态，上传的目标文件显示为绿色，不上传的为红色）

4）git commit -m “first commit”（first commit是自己为本次提交起的名字）

5）git remote add origin git@github.com:yourname/yourrepository.git（yourname是你在github上注册的用户名，yourrepository是你的远方存储卡，命令的最后记得加.git）

6）git push -u origin master（将本地库内的文件推送到github上对应关联存储库内）
